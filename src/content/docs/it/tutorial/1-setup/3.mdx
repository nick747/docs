## Modifica la tua home page

1. Nel tuo editor di codice, naviga nella sezione Explorer a `src/pages/index.astro` e cliccaci sopra per aprire il contenuto del file in una scheda modificabile.

    Il contenuto del tuo file `index.astro` dovrebbero apparire così:

    ```astro title="src/pages/index.astro"
    ---
    ---

    <html lang="en">
      <head>
        <meta charset="utf-8" />
        <link rel="icon" type="image/svg+xml" href="/favicon.ico" />
        <meta name="viewport" content="width=device-width" />
        <meta name="generator" content={Astro.generator} >
        <title>Astro</title>
      </head>
      <body>
        <h1>Astro</h1>
      </body>
    </html>
    ```

2. Modifica il contenuto del `<body>` della tua pagina.

    Inserisci nell'editor per modificare il testo del titolo sulla tua pagina e salva le modifiche.

    ```astro title="src/pages/index.astro" del={2} ins={3}
    <body>
      <h1>Astro</h1>
      <h1>My Astro Site</h1>
    </body>
    ```

3. Controlla l'anteprima sul browser e dovresti vedere il contenuto della tua pagina aggiornato col nuovo testo.

Congratulazioni! Ora sei uno sviluppatore Astro.

Il resto di questa unità ti preparerà per il successo con il controllo di versione e un sito web pubblicato che puoi mostrare.
